# AGENTS.md

This project uses agent tools and documentation located in the `.github` directory. For agent configuration, workflows, and compatibility information, please refer to the files in the `.github` folder.

If you are looking for agent-related setup, instructions, or integrations, see:

- `.github/AGENTS.md` (if present)
- `.github/workflows/` for automation workflows
- `.github/` for additional agent or CI/CD configuration files

If the `.github` directory does not exist, please create it and add your agent configuration files there for best compatibility with agent tools.
